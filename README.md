# 💳 Interactive Card Details Form

This is a solution to the **Interactive Card Details Form** challenge. It features a real-time card visualization and a highly interactive form built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## Technologies Used

* **Next.js 14** (App Router)
* **TypeScript** (Strict type checking)
* **Tailwind CSS** (Responsive and modern styling)
* **Framer Motion** (Smooth entry and state transition animations)

## Features

* **Real-time Synchronization:** Card visuals update instantly as the user types.
* **Smart Formatting:** Automatic card number grouping (4-digit spacing).
* **Robust Validation:** Checks for empty fields, incorrect number formats, and valid expiry dates.
* **Success State:** Animated confirmation screen upon successful form submission.
* **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens.
* **Active States:** Custom gradient borders on input focus to match the original design.
